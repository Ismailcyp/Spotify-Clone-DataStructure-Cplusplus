A feature-rich, console-based Music Player application built entirely in C++. This project simulates core music streaming functionalities—such as custom playlists, search histories, and play counts—by utilizing custom-built advanced data structures without relying on standard library containers like vectors.

---


## ✨ Key Features

### 🔐 User Management
* **Authentication:** Secure Login and Registration system. Each user's unique playlist and history are tied directly to their account session.

### 🎶 Playlist & Library Engine
* **Library Integration:** Browse a global library of songs and add them directly to your personal playlist via ID.
* **Custom Playlists:** Create, rename, and manage your playlist. Delete songs or manually add new custom tracks.
* **Smart Sorting:** Automatically sort your playlist ascending or descending based on a track's total play count.

### 🧠 Advanced Data Structures
* **Binary Search Tree (BST) Shuffling:** Implements a custom BST to shuffle the playlist using Preorder, Inorder, or Postorder traversals.
* **History & Tracking:** Tracks your "Recently Played" songs and "Search History" using custom Stack/Queue implementations.

### 💾 Data Persistence
* **File I/O:** The application state is fully persistent. Libraries, user credentials, and custom playlists are saved to and loaded from `.txt` files seamlessly upon booting and exiting.

## 🛠️ Built With

* **Language:** C++
* **Architecture:** Custom Data Structures (BST, Linked Lists, Arrays) & Object-Oriented Programming
* **IDE/Environment:** Code::Blocks / Command Line
